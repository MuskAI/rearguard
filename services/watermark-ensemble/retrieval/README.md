# Explicit Watermark Retrieval Index

Generated by `../build_retrieval_index.py` from public, labeled captures in
`wiltodelta/remove-ai-watermarks`.
The upstream project and copied capture crops are distributed under Apache-2.0;
see `UPSTREAM_LICENSE`.

- `index.faiss`: normalized 512-dimensional CLIP vectors, searched with inner product.
- `metadata.json`: platform, position, source path, crop geometry, and model identity.
- `calibration.json`: per-platform score thresholds, minimum margins, and position rules.
- `crops/`: the exact content-free or public corner crops represented by the vectors.

The index is evidence for known-platform attribution, not a universal watermark
classifier. Queries below threshold, with a small first/second-platform margin,
or in an incompatible position must return an unknown platform.
